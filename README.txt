*************************************************************************
INSTALLATION INSTRUCTIONS
*************************************************************************
MAC OSX:
	- Python is automatically installed with your OS. If for some reason
	it is not, download a Python 2.7.3 from http://www.python.org/getit/
	- Now install the pygame library from http://www.pygame.org/download.shtml
	Follow the instructions to install it in your python 2.7 directory.
	- Great, now you should be able to run the Pong.py file by simply
	using the command "python path/game-engine/Pong.py" from your terminal.
